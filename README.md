## Getting Started

1. **Clone the repository:**
   git clone https://github.com/KushalBamroliya/worldatlas.git
   cd worldatlas

2. **Install dependencies:**
   npm install

3. **Start the development server:**
   npm run dev

4. **Open in your browser:**
   http://localhost:5173
